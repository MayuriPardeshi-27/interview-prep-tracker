# My Interview Study Tracker

A simple web application to help you plan, track, and monitor your interview preparation topics.

## Features
- Add study topics with a title
- Set status: Pending, In Progress, or Completed
- Delete topics
- Track overall progress automatically
- Data saved in browser using localStorage

## How to Use
1. Open `index.html` in your browser
2. Type the topic you want to study
3. Select the status from the dropdown
4. Click "Add Study Topic"
5. Your topics and progress will be shown below

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage for saving data
